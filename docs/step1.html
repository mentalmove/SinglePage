Step 1



Step 1 does not use Javascript at all - it can be seen as preparation for the following.


The overlay holding the explanation is not part of the excercise; the technique used here is known as 'checkbox hack'. To prevent caching when the page is reloaded, attibute autocomplete="off" is used outside a form - tag; this leads (although working) to invalid HTML and will be corrected in one of the next steps.

To hide the overlay permanently for own experiments, the input - tag can be replaced by 

<input type="checkbox" class="hackbox close_explanation" autocomplete="off" id="close_explanation" checked>
The page holds three almost equal div - tags with a min-height of 100%; to make this having an effect, the parent element also needs a style property height (a consequence of prototypal behaviour: when no height is mentioned, obviously a non-zero height will be displayed while the property is still undefined).

Property position is set to relative; that enables inner elements to use the actual div as anchor.

Please notice the inner div having property padding-top at 25% (not 50%, what would seem more intuitive).
