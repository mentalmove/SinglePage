Scrolling up leads to the question:
Has the section previously recognised as visible gone out of view again?

That shall be translated into
Is the actual section's upper border below a defined limit, measured from the bottom?

Transformed to code, it results into 
        var actual = sections[stored_index];
        if ( (actual.y_position - scroll_y) > (window_height - SCROLL_UP_TOLERANCE) )
            stored_index--;

Now the task is almost completed. In the next step, code generation shall be replaced by loading a page;
this page's URL shall be shown in the browser address bar;
elements shall be stored to get more flexibility when extending the application.
