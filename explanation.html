The actual scroll position shall be compared with the section's position; therefore, each section's start position must be known. For suitable valuation of larger zones when scrolling up, the visible window height also should be known.

function get_window_height()
tries to access
window.innerHeight or document.body.clientHeight or document.documentElement.clientHeight
If none of these succeeds, a temporary div - tag will be written to the bottom right corner, and the upper distance to the body - tag will be measured.

function get_position(element)
detects an element's upper distance to its parent element, than sums the result to the parent element's upper distance to its parent element and so on until there is no more parent element.
Result will be an element's absolute position.
