Sometimes, Javascript is easier than CSS (and leads to even more valid code when unwanted browser cleverness shall be circumvented).

The overlay holds a simple span - tag working as a button; when clicked, it informs
function close_explanation()
about itself. close_explanation() forces the argument's parent node (i.e. the overlay) to disappear.

To hide the overlay permanently for own experiments,
display: none;
can be added to the properties of class explanation.
Alternatively, something like 
<script type="text/javascript">
            document.getElementsByClassName("explanation")[0].style.display = "none";
</script>
or 
<script type="text/javascript">
            close_explanation( document.getElementsByClassName("close_explanation")[0] );
</script>
could be written below the div - tag holding the explanation.
